## 🚀 Overview
A responsive single-page website that displays featured events in a clean and modern layout.


## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)


## ✨ Features
- Responsive design (mobile & tablet friendly)
- Dynamic event loading using JSON
- Search functionality to filter events by name
- Interactive Register button (UI-based)
- Clean and modern user interface


## ▶ How to Run

Or use Live Server for better performance.
